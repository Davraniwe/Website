Инновационное решение
для лечения алопеции

Разработка биологически активного препарата на основе активации сигнального пути Wnt/β-катенина для стимуляции роста волос

85% Эффективность
12 Месяцев разработки
0 Побочных эффектов

О проекте

Миссия

Создание эффективного и безопасного препарата для лечения всех типов алопеции на основе передовых научных исследований

Инновация

Активация сигнального пути Wnt/β-катенина для стимуляции стволовых клеток волосяных фолликулов

Результат

Крем для наружного применения с липосомной системой доставки для максимальной эффективности

Актуальность проблемы

Страдают от андрогенной алопеции к 70 годам

Испытывают проблемы с выпадением волос

Средняя эффективность существующих препаратов

Активация сигнального пути
